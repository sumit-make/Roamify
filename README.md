🚀 About The Project

WanderLust is a full-stack travel accommodation marketplace where homeowners can list properties and travelers can discover unique stays across destinations.

Built with a strong focus on scalability, clean architecture, and user experience, this project simulates a real-world platform like Airbnb.

🧠 Tech Stack
💻 Frontend
EJS (Dynamic templating)
HTML5, CSS3
JavaScript
⚙️ Backend
Node.js
Express.js
🗄️ Database
MongoDB
🔐 Authentication & Security
Passport.js
Sessions & Cookies
☁️ Cloud Services
Cloudinary (Image storage)
✨ Features

✔️ Secure Authentication System
✔️ Create, Edit & Delete Listings
✔️ Image Upload via Cloudinary
✔️ Review & Rating System
✔️ RESTful APIs
✔️ Flash Messages for Feedback
✔️ Fully Responsive UI
✔️ Clean MVC Architecture

🏗️ Architecture
MVC Pattern Used:

📁 Models       → Database schemas  
📁 Views        → EJS templates  
📁 Controllers  → Business logic  
📁 Routes       → API handling

⚙️ Getting Started
🔽 Clone Repository
git clone https://github.com/your-username/wanderlust.git
cd wanderlust
📦 Install Dependencies
npm install
🔐 Setup Environment Variables

Create .env file:

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret

DB_URL=your_mongodb_uri
SESSION_SECRET=your_secret
▶️ Run App
npm start

🌐 Open: http://localhost:3000
